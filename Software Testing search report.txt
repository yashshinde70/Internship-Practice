1. What is Software Testing?

Software Testing is the process of verifying and validating software applications to ensure they meet requirements and are free from defects.

3. Automation Testing Frameworks
A. Data-Driven Framework

Used When: Testing multiple sets of data.

Example:
Login functionality with multiple usernames and passwords.

B. Keyword-Driven Framework

Used When: Non-technical testers participate in automation.

C. Hybrid Framework

Used When: Combines Data-Driven and Keyword-Driven approaches.

Most Common in Industry

D. Page Object Model (POM)

Used When: Large web applications.

Benefits:

Better maintenance
Reusable code
Easy scalability
E. BDD Framework

Tools:

Selenium
Cucumber
Gherkin

Used For:
Business-readable test scenarios.
